A simple ODM library for Firebase Cloud Firestore. Read the example app to understand more. This library is not stable enough to use for production code. Just testing for fun :3 **M**ost **V**ui_t√≠nh **P**erson :))
# Usage
### Gradle
Add this to your module's `build.gradle` file:
```
dependencies {
        // ... other dependencies
        implementation 'cf.bautroixa.firestoreodm:firestoreodm:0.0.1'
}
```
### Maven
```
<dependency>
    <groupId>cf.bautroixa.firestoreodm</groupId>
    <artifactId>firestoreodm</artifactId>
    <version>0.0.1</version>
    <type>pom</type>
</dependency>
```
